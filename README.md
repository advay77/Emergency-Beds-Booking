## Emergency-Beds-Booking

Emergency-Beds A web page to be integrated in a website to book hospital beds simply by filling a form and seeing the hospital description 🙂✌🏻.

## Tech Stack
1. React + Vite
2. TypeScript
3. Leaflet.js (For Maps)
4. Tailwind CSS
5. A bit files of .json and Components (config.json, package.json, app.json)
6. Node.js
7. JavaScript and React Icons
